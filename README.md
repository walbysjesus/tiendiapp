# TiendiApp 🛒

Plataforma web progresiva (PWA) para gestión de pedidos, inventario y ventas en tiendas de barrio.

## 🚀 Tecnologías
- PWA (HTML, CSS, JavaScript)
- Firebase (Autenticación y Base de datos)

## 📱 Características
- Instalable como app
- Funciona en celular y computador
- Gestión de pedidos
- Control de inventario
